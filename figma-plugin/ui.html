📋 表格欄位填寫規則 ▾
基本原則
・每一列資料 → 各佔一行（換行分隔）
・每一欄資料 → 用 Tab 或 兩個以上的空格 分隔
・行尾多餘的空格/Tab 會自動忽略

第一欄（標籤欄）要留空時
・在該行開頭加一個 Tab 或兩個空格
・例如：[Tab]DENOMINATION → 第一欄空白，第二欄顯示 DENOMINATION

跨欄 Header（自動偵測）
・若某列的欄位數比其他列少，最後一個值會自動延展填滿剩餘欄位
・例如：只填 [Tab]DENOMINATION，DENOMINATION 會自動跨越所有 Denom 欄

注意事項
・欄位值本身不能含有連續兩個以上的空格，否則會被誤判為欄位分隔
